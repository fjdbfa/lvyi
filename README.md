# 项目总体分析报告

## 1. 项目基本信息

### 1.1 项目概述
- **项目名称**: demo
- **项目类型**: Spring Boot 多模块项目
- **项目描述**: 一个集成了AI对话、论坛系统、任务管理、电商功能的综合性后端服务

### 1.2 技术栈版本

| 技术组件 | 版本号 |
|---------|--------|
| **JDK** | Java 17 |
| **Spring Boot** | 3.5.6 (子模块) / 3.4.3 (父模块) |
| **MyBatis-Plus** | 3.5.5 |
| **MySQL Connector** | Runtime (跟随Spring Boot版本) |
| **Redis** | Spring Boot Data Redis |
| **JWT** | 0.11.5 |
| **Lombok** | 1.18.30 |
| **Hutool** | 5.8.20 |
| **Aliyun OSS** | 3.17.4 |
| **LangChain4j** | 0.34.0 |
| **Jackson** | Spring Boot 默认版本 |
| **FastJSON** | 1.2.83 |
| **OkHttp** | 4.9.3 |
| **JavaMail** | 1.6.2 |

### 1.3 项目架构

#### 模块划分
```
demo (父模块)
├── pojo      - 实体层：定义实体类、DTO、VO
├── common    - 公共层：工具类、常量、异常处理、结果封装
└── server    - 服务层：控制器、服务、Mapper、配置
```

#### 分层架构
```
Controller → Service → Mapper → Database
     ↓          ↓
   DTO/VO     Entity
```

#### 设计模式
- **三层架构**: Controller-Service-Dao
- **依赖注入**: Spring IoC容器管理
- **AOP切面**: 拦截器、异常处理
- **RESTful API**: 标准HTTP方法设计

## 3. 数据存储方案

### 3.1 关系型数据库
- **MySQL**: 主要数据存储
- **MyBatis-Plus**: ORM框架
- **表数量**: 约15张

### 3.2 缓存数据库
- **Redis**: 
  - 验证码缓存（60秒过期）
  - Session管理
  - 连接池配置: max-active=10

### 3.3 对象存储
- **阿里云OSS**: 图片文件存储
  - Bucket: tz-81-test
  - 地域: 杭州数据中心
